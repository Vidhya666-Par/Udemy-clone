# Udemy-clone
"A responsive Udemy Clone website built using HTML and CSS, featuring course listings, categories, search functionality, and a modern user interface."
- Responsive Navbar
- Search Bar UI
- Course Categories
- Recommended Courses Section
- Popular Courses Section
- Footer Section

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Font Awesome

## Files
- `index.html` – Main webpage structure
- `style.css` – Styling and responsive design

## How to Run
1. Download the project files.
2. Open `index.html` in your browser.
